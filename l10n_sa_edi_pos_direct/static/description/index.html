🎯 Overview / نظرة عامة

An optimized replacement for the standard l10n_sa_edi_pos module, providing direct ZATCA integration with significant performance improvements and enhanced functionality designed specifically for high-volume retail environments in Saudi Arabia.

موديول محسن بديل عن l10n_sa_edi_pos الافتراضي، يوفر تكامل مباشر مع هيئة الزكاة والضريبة مع تحسينات كبيرة في الأداء ووظائف محسنة

🔄 Module Comparison / مقارنة الموديولات

Feature / الخاصية l10n_sa_edi_pos (Standard / الافتراضي) l10n_sa_edi_pos_direct (Enhanced / المحسن)
Customer Data Requirement / متطلبات بيانات العميل ❌ Requires full customer data for every POS transaction
يتطلب بيانات العميل كاملة مع كل عملية 		✅ Cash customer support (no customer data needed)
دعم العميل النقدي (لا يتطلب بيانات العميل)

ينشئ فاتورة PDF مع كل طلب 		✅ No PDF generation - POS receipts only

تأثير قليل على قاعدة البيانات - معالجة على واجهة نقاط البيع فقط

🚀 Installation & Configuration

Remove Standard Module

Uninstall the existing l10n_sa_edi_pos module from your system to avoid conflicts.

Install Enhanced Module

Install l10n_sa_edi_pos_direct through the Apps menu or via command line.

Verify Configuration

Ensure your ZATCA certificate is properly configured in the invoice journal settings.

Enable Direct Mode

Navigate to POS Configuration and enable "ZATCA Direct Mode" for your point of sale.

Test Transactions

Perform test transactions to verify QR code generation and ZATCA submission workflow.
